# Standards

- Use prompt-provided AGENTS instructions as active project instructions; no tracked `AGENTS.md` exists.
- Use `rg` for source search.
- Use `apply_patch` for manual edits.
- Keep root `Cargo.toml` read-only because it is generated.
- Keep procurement/user deliverables free of internal workflow notes unless explicitly requested.
- Record assumptions as assumptions.
- Treat scanner and strategist reports as findings, not commands; verify before implementing.
- Preserve existing root/user diffs.
- Never claim an acceptance criterion passed unless it was implemented and verified.
